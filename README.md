# Prove AI POC

Proof of Concept application for Prove AI.

## Setup

1. Install dependencies:
```bash
npm install
```

2. Start the server:
```bash
npm start
```

3. Open your browser to:
```
http://localhost:3005
```

## Project Structure

- `server.js` - Express server
- `public/` - Frontend files
  - `index.html` - Main HTML file
  - `styles.css` - Styles
  - `script.js` - JavaScript
